# Tic-Tac-Toe

**Target time:** 30 min

## Problem
NxN Tic-Tac-Toe with win/draw/invalid detection.

## Requirements
- NxN board
- alternate marks; reject occupied/out-of-bounds
- detect row/col/diag win and draw

## Passing criteria
- win detection NxN
- draw detection
- invalid move rejected
