# Padhai kaise hoti hai — ek jagah, poora flow

> **Sab kuch isi workspace mein.** Bahar links dhundhne ki zaroorat nahi.  
> MODULE.md = **textbook (theory)** · `practice/` = **exercises** · Chat = **doubt + code check**

## 2 jagah + chat

```
┌──────────────────────────────────────────┐
│  MODULE.md  — THEORY (yahin padho)       │
│  Visual map → Theory sections → Recall   │
└──────────────────────────────────────────┘
                    ↓
┌──────────────────────────────────────────┐
│  practice/  — CODE (yahin likho)         │
│  Stubs with gaps · Passing criteria      │
└──────────────────────────────────────────┘
                    ↓
┌──────────────────────────────────────────┐
│  Cursor CHAT — sirf jab chahiye          │
│  Doubt · recall drill · code review      │
└──────────────────────────────────────────┘
```

## Ek session (60–90 min)

| Step | Kya | Kahan |
|------|-----|-------|
| 1 | Visual map | MODULE.md top |
| 2 | **Theory** padho — section by section | MODULE.md `## Theory` |
| 3 | **Practice** — stub complete karo | `modules/XX/practice/` |
| 4 | Doubt / code check | Cursor chat `@MODULE.md` |
| 5 | Redraw challenge | Excalidraw → NOTES |
| 6 | NOTES update | `NOTES.md` |

## Coach (Cursor chat) kab use karo

| Haan | Nahi |
|------|------|
| "Yeh paragraph samajh nahi aaya" | Poori theory dubara likhwana |
| "Mera code check kar" | Assignment ka poora solution maangna |
| Active recall drill | Link list maangna |
| "A2 mein error aa raha" | Vibe code karwana |

## Cursor ka asli fayda (tera use case)

1. **Theory** — MODULE.md mein likhi hai, Obsidian mein padho  
2. **Practice** — tum code likho, `practice/` mein  
3. **Teacher** — chat mein doubt + review + recall  

Links optional hain — appendix mein, zaroorat ho tab.
