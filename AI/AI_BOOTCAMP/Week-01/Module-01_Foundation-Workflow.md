# Module 01 - Foundation Workflow (Execution Mindset)

## Why this module?
$100k+ role ke liye sirf model banana enough nahi hai. Tumhe "reliable builder" banna hai jo production constraints samjhe.

## Core Ideas
1. **Project-first learning**  
   Theory ko immediately code output mein convert karo.
2. **Signal over noise**  
   Certificates nahi, shipped repositories and measurable impact matter karta hai.
3. **System > Motivation**  
   Daily ritual tumhe carry karega, mood nahi.

## Hiring Signals to Build from Week 1
- Clean repo structure
- Repeatable run steps
- Data checks and assumptions listed
- Small but testable scripts
- Clear explanation of trade-offs

## Execution Stack (Phase 0)
- Python
- SQL
- Git/GitHub
- Linux terminal basics
- Basic logging and data validation

## Deliberate Practice Loop (Use Daily)
- Step 1: concept read
- Step 2: closed-book summary
- Step 3: code attempt
- Step 4: bug log
- Step 5: refactor + commit note

## Active Recall Prompts
- "Notebook prototype aur production script mein kya farq hai?"
- "Model accuracy high ho sakti hai but business value low kaise ho sakti hai?"
- "Data quality checks ke bina ML build karna risky kyun hai?"

## This Week Focus Question
"Kya main raw data se business-ready table bana sakta hoon consistently?"
